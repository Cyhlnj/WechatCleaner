# 微信资源清理器

一个运行在浏览器里的微信资源清理小工具，用来扫描、预览、筛选、查重、导出并清理微信聊天资源目录中的图片、视频和文件。

它是一个纯前端静态页面，不需要后端服务，不会把文件上传到服务器。所有扫描、预览、Hash 计算和删除操作都在本机浏览器内完成。

## 功能特性

- **本地目录扫描**：通过浏览器目录授权读取微信资源文件夹。
- **资源自动识别**：按图片、视频、普通文件分类展示。
- **卡片式预览**：图片和视频可直接预览，文件显示扩展名、日期和大小。
- **筛选与排序**：支持按类型、最小大小、时间、名称和文件大小筛选排序。
- **分页浏览**：默认每页展示 200 个资源，适合处理大量文件。
- **批量选择与清理**：支持选择当前页、清空选择、批量删除选中文件。
- **单文件直接删除**：每个资源卡片都可以单独删除。
- **重复文件查找**：
  - 快速疑似：按文件大小、类型、扩展名和视频时长判断。
  - 精确 Hash：按 SHA-256 内容哈希判断。
- **重复文件辅助勾选**：可按“每组保留最新，选中其余”快速标记待清理项。
- **清理清单导出**：删除前可导出 CSV，方便回查。
- **进度可中断**：扫描或查重过程中可以随时停止。
- **偏好记忆**：自动记住主题、筛选条件、排序方式、删除确认设置和上次授权目录。
- **深色模式**：支持跟随系统、浅色和深色主题。

## 适用场景

- 微信聊天图片、视频和文件长期积累，占用大量磁盘空间。
- 想先可视化查看资源，再手动决定删除哪些文件。
- 想找出重复图片、重复视频或重复文件，减少无效占用。
- 想在删除前导出一份清理清单，降低误删风险。

## 浏览器要求

推荐使用最新版：

- Google Chrome
- Microsoft Edge

本工具依赖浏览器的 [File System Access API](https://developer.mozilla.org/en-US/docs/Web/API/File_System_API) 来选择目录和删除文件。Safari、Firefox 或其他不支持该 API 的浏览器可能无法直接选择目录或删除文件。

## 快速开始

1. 下载或克隆项目到本地。
2. 使用 Chrome / Edge 打开 `index.html`。
3. 点击页面里的 **选择文件夹**。
4. 手动选择微信资源目录，例如：

   ```text
   ~/Library/Containers/com.tencent.xinWeChat/Data/Documents/xwechat_files/<你的微信数据目录>/msg
   ```

   也可以选择更具体的子目录，例如：

   ```text
   ~/Library/Containers/com.tencent.xinWeChat/Data/Documents/xwechat_files/<你的微信数据目录>/msg/video
   ```

5. 等待扫描完成后，根据类型、大小、排序、重复文件等条件筛选。
6. 删除前建议先点击 **导出清理清单**。
7. 确认无误后，再删除选中文件。

## 使用说明

### 选择目录

浏览器不能主动读取你电脑上的固定绝对路径，必须由你手动选择文件夹并授权。

首次选择目录后，页面会尝试记住该目录授权。下次打开页面时，如果浏览器仍保留权限，会自动恢复并扫描；如果需要重新授权，点击 **恢复上次文件夹** 即可。

### 筛选和排序

页面支持：

- 类型：全部、图片、视频、文件
- 最小大小：不限、≥ 1 MB、≥ 5 MB、≥ 10 MB、≥ 50 MB、≥ 100 MB
- 排序：大小、时间、名称
- 显示：只看已选

这些设置会自动保存到浏览器本地。

### 查找重复文件

点击 **查找疑似重复文件** 前，可以选择查重方式：

- **快速疑似**：速度更快，适合先粗筛。图片和文件主要按类型、扩展名和大小判断；视频会额外参考时长。
- **精确 Hash**：结果更准确，会读取文件内容并计算 SHA-256，文件较多或较大时会更慢。

进入重复模式后，可以点击 **每组保留最新，选中其余**。这个操作只会帮你勾选文件，不会自动删除。

### 删除文件

支持两种删除方式：

- 单个资源卡片上的 **直接删除**
- 页面顶部的 **删除选中文件**

默认开启“删除前二次确认”。你也可以在页面里关闭确认开关，但不建议在不熟悉目录内容时关闭。

> 注意：浏览器执行删除时，通常不是移动到 macOS 废纸篓，而是直接删除。请谨慎操作。

### 导出清理清单

点击 **导出清理清单** 会下载一个 CSV 文件，包含：

- 文件名
- 相对路径
- 类型
- 文件大小
- 最后修改时间

建议在批量删除前先导出，方便后续核对。

## 复制完整路径

预览弹窗里可以点击 **复制完整路径**。

由于纯静态网页无法直接读取用户所选目录的真实绝对路径，完整路径会基于 `app.js` 里的 `DEFAULT_ROOT_PATH` 拼接生成。

如果你自己的微信资源目录不同，可以按需修改：

```js
const DEFAULT_ROOT_PATH = '/你的/微信资源/目录';
```

## 安全说明

- 本项目没有后端服务，不会上传你的文件。
- 文件内容只在本机浏览器中读取，用于预览、统计大小或计算 Hash。
- 删除操作由浏览器文件系统能力执行，删除后可能不可恢复。
- 建议先在小目录中测试，确认符合预期后再处理大目录。
- 建议删除前导出清理清单，必要时提前备份重要文件。

## 常见问题

### 为什么必须手动选择文件夹？

浏览器出于安全限制，网页不能直接访问电脑上的任意路径。只有你主动选择并授权的目录，网页才能读取。

### 为什么 Safari / Firefox 不能用？

本工具依赖 File System Access API。不同浏览器支持情况不同，Chrome / Edge 支持最好。

### 为什么完整路径可能不对？

浏览器只提供所选目录下的相对路径，不提供真实绝对路径。因此完整路径需要通过 `DEFAULT_ROOT_PATH` 手动配置。

### 删除后还能恢复吗？

不一定。浏览器删除通常不会进入系统废纸篓。重要文件请先备份。
